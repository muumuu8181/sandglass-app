# ⏳ 砂時計シミュレーター

Matter.jsを使用した物理演算による砂時計シミュレーターです。

## 🌟 機能

- **物理演算**: Matter.jsによるリアルな砂の動き
- **反転機能**: ワンクリックで砂時計を180度回転
- **タイマー**: 経過時間の自動計測
- **リセット**: 初期状態に戻す
- **レスポンシブデザイン**: PC・スマホ両対応

## 🎮 使い方

1. ブラウザでindex.htmlを開く
2. 砂が自動的に落下開始
3. 「反転」ボタンで砂時計を回転
4. 「リセット」ボタンで初期状態に戻す

## 📱 動作環境

- Chrome, Firefox, Safari, Edge
- スマートフォン対応

## 🛠️ 技術スタック

- Matter.js (物理演算)
- Canvas API (描画)
- Firebase (フィードバック記録)
